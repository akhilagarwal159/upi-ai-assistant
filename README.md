# 💳 UPI AI Assistant

## 🚀 Overview
A smart fintech assistant that helps users understand UPI transactions, failures, and spending patterns.

## ✨ Features
- Explains failed transactions with reasons
- Suggests next steps
- Provides spending insights
- Visualizes merchant-wise spend

## 🛠️ Tech Stack
- Python
- Streamlit
- Plotly

## 💡 Problem Statement
UPI failures often confuse users and increase support load. This tool simplifies understanding and improves user experience.

## 🔮 Future Scope
- AI-powered insights
- Fraud detection
- Smart financial recommendations
